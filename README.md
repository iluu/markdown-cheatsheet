## Text formatting

  Common text

  _italics_ or *italics*

  __bold__ or **bold**

  ___bold and italics___ or ***bold and italics***

## Links

  [Named Link](http://www.google.fr/)

  http://www.google.fr/ or <http://example.com/>

  [Ref Link]
  [Ref Link]:http://www.google.com
